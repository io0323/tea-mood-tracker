# TeaMood Tracker

TeaMood Tracker is an Android life-log app built with Kotlin and Jetpack Compose.

## Features
- Log tea intake, mood, and time of day.
- Review today's mood and caffeine estimate.
- Browse history and weekly caffeine visualization.

## Tech stack
- Kotlin
- Jetpack Compose (Material3)
- MVVM + simple Clean Architecture
- Local in-memory repository (Room-ready structure)
